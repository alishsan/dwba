## Pre-Submission Tasks

1. **Review manuscript** for:
   - Grammar and spelling
   - Technical accuracy
   - Clarity of presentation
   - Completeness of references

2. **Prepare cover letter** highlighting:
   - Significance of the work
   - Novel contributions
   - Relevance to computational physics community

3. **Check CPC guidelines**:
   - Visit: https://www.sciencedirect.com/journal/computer-physics-communications/publish/guide-for-authors
   - Verify all requirements met
   - Check page limits (typically 15-20 pages for regular papers)

4. **Prepare author information**:
   - Full names and affiliations
   - Email addresses
   - ORCID IDs (if available)
   - Corresponding author clearly marked

## Submission Process

1. **Online submission**:
   - Go to: **https://www.editorialmanager.com/comphy/default2.aspx**
   - **Important**: This is the correct link for Computer Physics Communications
   - Create account (if new)
   - Follow submission wizard
   - Verify you're on the Computer Physics Communications site (not other journals)

2. **Upload files**:
   - Main LaTeX file
   - PDF version (if required)
   - Figures (if any)
   - Supplementary material (if any)

3. **Complete submission form**:
   - Select article type (Research Paper)
   - Enter all author information
   - Upload cover letter
   - Suggest reviewers (optional but helpful)

4. **Submit and track**:
   - Submit manuscript
   - Receive confirmation email
   - Track status through Editorial Manager
